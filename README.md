# controle-de-gastos
sistema simples de controle financeiro desenvolvido em Go para gerenciar gastos mensais e acompanhar o saldo do salario.

## sobre o projeto
este programa permite que o usuario cadastre seus gastos mensais, visualize um resumo financeiro e  gerencie de forma pratica. O sistema calcula automaticamente quanto sobra do salario apos cada gasto registrado

## funcionalidades
- **definir salario mensal: configure quanto ganha por mes(crucial para o programa funcionar corretamente)
- **adicionar gastos: registre o nome e valor de despesas mensais
- **ver gastos: veja a lista completa de gastos e seus valores
- **redefinir salario: atualize o valor de salario quando necessario

## como executas
1. certifique-se de ter o Go instalado
2. clone este repositorio
3. navegue ate a pasta do projeto
4. execute 'go run main.go'

## como usar
1. primeiro, defina seu salario(opcao 3)
2. adicione seus gastos mensais(opcao 1)
3. visualize o resumo e saldo restante(opcao 2)
4. para sair escolha a opcao 0

## tecnologias utilizadas
- Go (Golang)
